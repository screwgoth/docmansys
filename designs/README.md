## Viewing the Mockups

All mockups are standalone HTML/CSS files. To view them:

1. **Clone the repo and checkout the design branch:**
   ```bash
   git clone https://github.com/screwgoth/docmansys.git
   cd docmansys
   git checkout design
   ```

2. **Open the HTML files in your browser:**
   ```bash
   cd designs/mockups
   open dashboard.html      # macOS
   # or
   xdg-open dashboard.html  # Linux
   # or just drag them into your browser
   ```

## Files

- **`dashboard.html`** - Main dashboard with stats and activity feed
- **`documents.html`** - Document grid view with search/filter
- **`upload.html`** - File upload interface with drag-and-drop
- **`viewer.html`** - Document viewer with metadata sidebar
- **`index.html`** - Landing page design
- **`DESIGN_PRESENTATION.md`** - Full design rationale and decisions

## Design Highlights

✨ **Modern & Clean** - Card-based layouts, ample whitespace  
🎨 **Professional Palette** - Indigo/purple gradients  
📱 **Responsive Grid** - Adapts to screen sizes  
♿ **Accessible** - High contrast, semantic HTML  
🎯 **User-Focused** - Intuitive navigation and clear CTAs
